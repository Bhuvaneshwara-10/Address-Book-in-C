# Address Book Management System in C

##  Project Overview

The Address Book Management System is a console-based application developed in the C programming language. This project allows users to store, manage, and organize contact details efficiently using file handling and structured programming concepts.

The application supports operations such as adding, searching, editing, deleting, and viewing contacts. It demonstrates practical implementation of core C concepts including structures, arrays, functions, pointers, dynamic memory handling, and file operations.

---

##  Features

* Add new contacts
* Search contacts by name, phone number, or email
* Edit existing contact details
* Delete contacts
* View all saved contacts
* Sort contacts alphabetically
* Save contacts permanently using file handling
* User-friendly menu-driven interface

##  How to Compile and Run

### Using GCC Compiler

```bash
gcc *.c -o addressbook
./addressbook
```

### On Windows

```bash
gcc *.c -o addressbook.exe
addressbook.exe
```
